Resize and/or change brightness

Is the background image too large or too bright?


Of course you are able to resize and/or change the brightness of the background image.


Resize

Please do not work with a 'monster' image or with a thumbnail as backglass background.

My recommendation is: Your backglass background image should be between 1920 x 1080 pixel and 1024 x 768. DO NOT USE smaller source images. If possible start with a pretty large source image and resize it to a matching size but don't change the aspect ratio.


I.e. I started at 'SORCERER' with a source image of 2382 x 1506 and resized it to ...

... a size of 70%. The result is an background image for the backglass with 1667 x 1054 pixels.


In our "Bally Game Show" project there is no need to resize the starting image as we are at the very low level of 1024 x 768. It's no perfect image for our backglass.



Change brightness

Why is it important to change the brightness?

Normally the source image is bright, too bright. To get some great flashing, some standard bulbs or just a soft illumination we need to darken the image at design startup. It's funny, right now you darken the image and in a few minutes you light it up. But that's how it works.


Our "Bally Game Show" source image is really bright. We have to darken it a lot, try a value of 60%.



Pretty important, so once again:

As I mentioned earlier your source image is not touched of all the changes you are doing to the designer's images. The designer is working with its own copies of the images and is storing these resized or darkened images into the saved project file.
