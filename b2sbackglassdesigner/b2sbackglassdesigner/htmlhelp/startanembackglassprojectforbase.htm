These are the settings for Gottlieb's 'Baseball', an EM table from 1970.


  • 'Project name' is just the way I like it.
  • 'Backglass (file) name' is a new name since we have to do some VP script changes and need a new table name too,
  • 'Table type' is pretty clear.
  • 'Number of players' is '1' since it is a one player table.
  • 'Communication mode' is the B2S internal communication mode since this is an EM table and there is no VPinMAME running.
  • 'Destination type' is what I called output type earlier. I have chosen a 'directB2S' backglass since right now I don't want to code anything with Visual Studio 2010.
  • 'Create EM default code ...' is disabled in this constellation. More infos later.
  • 'Count of B2S data elements' is uninteresting here.
  • ''B2S DMD' location setting' is set to 'No B2S DMD'.
