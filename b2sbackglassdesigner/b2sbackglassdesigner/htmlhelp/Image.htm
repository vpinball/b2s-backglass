Tool bar and menu items ››

Import images


Import backglass image

Imports an external backglass image. The standard Windows 'file open dialog' will be opened to select an image file.


Reload backglass image

Reloads the selected backglass image if possible. If the current backglass image file could not be found an error message pops up.


Import illumination image

Imports an illumination backglass image. The standard Windows 'file open dialog' will be opened to select an image file.
Please notice: All illumination will be taken from this illumination image. No way to use the designer's illumination.


Import DMD image

Imports a 'B2S DMD' image. The standard Windows 'file open dialog' will be opened to select an image file.
Not available when you have selected 'No 'B2S DMD'' for ''B2S DMD' location setting' at the backglass settings.


DMD area


Copy DMD image from backglass

Shows a yellow frame to select the copied area of the backglass image. This part of the backglass will be used as the 'B2S DMD' image. Click on the yellow camera icon to create the image.

Not available when you have selected 'No 'B2S DMD'' for ''B2S DMD' location setting' at the backglass settings.


Set default DMD location

Starts the 'Set DMD location' mode and shows a red rectangle in the size of the added 'B2S DMD' image.

Not available when you have selected 'No 'B2S DMD'' for ''B2S DMD' location setting' at the backglass settings.


Grill height


Set grill height

Starts the 'Set grill height' mode and shows a green line as the 'grill height marker'. Move the top of this line to the top location of the grill and click to set the grill height. To deactivate the 'Set grill height' without setting or changing the grill height mode click this menu item once again.


Set mini grill height

Starts the 'Set mini grill height' mode and shows a red line as the 'mini grill height marker'. Move the top of this line to the top location of the mini grill above the standard grill height and click to set the mini grill height. To deactivate the 'Set mini grill height' without setting or changing the mini grill height mode click this menu item once again.

Check my 'The Addams Familiy' backglass  to get the background of a 'mini grill'.


Resize

Opens the 'Resize' dialog window to resize the current backglass or DMD image by percentage or by absolute size (in pixel). For more information please click here.

Notice that you resize the internal backglass image of the designer and not the source image in the file system.


Brightness

Opens the 'Brightness' dialog window to change the brightness of the current backglass or DMD image by percentage. For more information please click here.

Notice that you change the brightness of the internal backglass image of the designer and not of the source image in the file system.
