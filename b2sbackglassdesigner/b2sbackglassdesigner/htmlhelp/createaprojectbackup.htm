Create a project backup

Creating a first B2S backglass ›› More interesting designer stuff ››

Do you want to test some weird illumination but you don't want to loose your current backglass state? Of course no problem, create  a backup of your current state. You can return to this backup state whenever you want to.


Open an already existing backglass project, maybe 'Close Encounters of the Third Kind' once again.


Start to create a backup of your current backglass project with

  • 'File' -> 'Save backup as ...'


A  pretty simple dialog will be opened. Enter a backup name to identify your backup, maybe like this ...


... and click 'Ok'. Your current project is now saved as a designer project backup file with the extension 'b2b'.


Now you're ready to do all the changes in your current backglass project you want to try. It's even okay to save your project since you're able to return to the backup state.


Okay, all your changes are done but you don't like them and you want to return to the backup state. Open your backup with

  • 'File' -> 'Open backup ...'


You get a standard Windows file open dialog limited to '*.b2b' files. Look for your backup file and select it. It will be opened like a standard project but of course it's still the backup of your project. The interesting stuff is happening when you save this backup project once again. Then you get this question ...

(translation of the buttons is 'Yes', 'No', 'Cancel' :-))


Click on ...

  • 'Yes' ('Ja') to save this backup file once again
  • 'Cancel ('Abbrechen') to cancel the whole process
  • 'No' ('Nein') to get this next message box ...

    Pretty clear what this means, isn't it? Click on ...
    • 'Yes' to save it a your standard project and overwrite your last standard backglass project state
    • 'No' to cancel this
    • 'Cancel' to cancel the whole process
