Save the backglass project

Creating a first B2S backglass ›› A B2S backglass from scratch ››

There are a few ways to save your backglass project.

  • Click on the 'Save' tool button in the toolbar.
  • Click on the menu item 'File' -> 'Save'.
  • Click on the menu item 'File' -> 'Save all'.



Please notice:

All data is stored in an '*.b2s' file in a folder 'Projects' in the backglass designer EXE folder. All involved images are stored in this file too. This means all changes you're doing to the images do not touch the source images.

There is happening  more during the saving. All imported images are copied to a project sub folder in the 'Projects' folder. And the recent project list is updated.


There are a few more 'Save' menu items under 'File'. You are able to create a copy of an already existing project or create a backup to try some changes and maybe return to a dedicated project state. More infos later.
