Tool bar and menu items

Here are the tool bar items.



  • 1 - New
    Starts a new backglass project and opens the settings dialog. For more infos click here.
  • 2 - Open
    Opens the 'file open dialog' to select a already existing project file and open the project.
  • 3 - Save
    Saves the current selected project.
  • 4 - Undo and Redo
    Not fully supported right now. Some of the Undos are working already, especially to undo frame moving and deleting.
  • 5 - Zoom
    Zooms the backglass or DMD image. Use 'Window' to fit the image to the current window.
  • 6 - Import background image
    Opens the 'file open dialog' to select a backglass image as main background image.
  • 7 - Show reel or LED frames
    Shows or hides all already added reel and/or LED frames. Frames can be moved, resized, ...
  • 8 - Add new reel or LED frame
    Adds a new reel or LED frame.
  • 9 - Show reels or LEDs
    Shows or hides all already added reel and/or LED displays. Displays are not editable.
  • 10 - Show illumination frames
    Shows or hides all already added illumination frames. Frames can be moved, resized, ...
  • 11 - Add new illumination frame
    Adds a new illumination frame.
  • 12 - Select lamp filter
    Filters the visible illumination frames.
  • 13 - Show illumination with fix intensity '2'
    Shows the filtered illumination in a fix intensity of level '2' for each illumination source.
  • 14 - Show illumination with accurate intensity
    Shows the filtered illumination in accurate intensity.
  • 15 - Select current image
    Switches between backglass and DMD image.
  • 16 - Help
    Opens the help manual.
