The designer for the first time

Creating a first B2S backglass ›› A B2S backglass from scratch ››

When you've opened the designer for the very first time it looks a bit 'naked'. Of course, there is nothing to be shown since there is no backglass project opened. What you see is the empty main window. Here is happening most of the designing. We start in a minute.


This is the designer's toolbar. Move your mouse over the icons and you get some tooltip info. More infos later.


Beside the main window there are four tool windows. To open them use the 'Window' menu item. When it's necessary two of them will pop up automatically. It makes sense to have a look at these tool windows right now:


  • 'Resources'
    Open this tool window with 'F5'. This window is showing the basic resources involved in the design process. I.e. there are thumbnail previews for all imported background or DMD images.


  • 'Reels & LEDs'
    Open this tool window with 'F6'. This window is showing all infos about a 'score'. It will pop up automatically if necessary. More infos later.


  • 'Illumination'
    Open this tool window with 'F7'. This window is showing all infos about a 'bulb'. It will pop up automatically if necessary. More infos later.


  • 'History'
    Open this tool window with 'F8'. This window is showing a history of your changes. You can undo a lot of your changes especially all moving or resizing actions. But please notice, right now some of the actions are not included in the UNDO mechanism.
