Tool bar and menu items ››

New ...

Starts a new B2S backglass project. The settings dialog will be opened automatically.


Open ...

Opens an already created backglass project. The standard Windows 'file open dialog' will be opened to select the '*.b2s' backglass project file.


Open recent

The last 10 opened projects will be listed here for a pretty fast 'project entry'.


Open backup ...

Opens a 'backuped' backglass project. The standard Windows 'file open dialog' will be opened to select the '*.b2b' backglass backup file. An opened project backup can be saved as a  backglass project backup or resaved as a standard backglass project.


Close

Closes the currently selected backglass project.


Save

Saves the currently selected backglass project.


Save backup as ...

Closes the currently selected backglass project as a backglass project backup. Does not overwrite the standard backglass project state.


Save copy as ...

Saves the currently selected backglass project as a copy. Perfect to start a new project matching an already existing backglass project. The settings dialog will be opened automatically.


Save all

All opened backglass projects will be saved.


Settings ...

Enter the general backglass settings. For more information please click here.


Import 'directB2S' backglass file ...

Imports a 'directB2S' backglass file into the designer.


Exit

Get out.
