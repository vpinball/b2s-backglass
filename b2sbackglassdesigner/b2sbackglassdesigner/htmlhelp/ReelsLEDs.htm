Reels & LEDs

Tool bar and menu items ››

Choose reel or LED type ...

Opens the dialog to select reels or LEDs. Recommendation for LEDs are the 'Rendered LEDs'. For reels you have to select 'Image sets'. You can import your own image sets. Select the type by clicking on it and saving with 'Ok'. For more information please click here.


Show reel or LED frames

Shows or hides the already added score display frames in edit mode.


Add new reel or LED frame

Adds a new score display frame. Select the reel or LED type of this score display by clicking on a reel or LED type at the 'Reels & LED' tool window.


Show reels or LEDs

Shows or hides the already added score display frames in read-only mode.
