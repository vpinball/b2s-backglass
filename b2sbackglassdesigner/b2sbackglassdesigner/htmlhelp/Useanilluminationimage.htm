Well, I don't think so, I don't expect it, this is truly unthinkable, but maybe, maybe, maybe, ...


... you don't like the illumination the designer is doing.


No problem. Do all the painting in some painting software outside the designer. Start with your standard dark background image and draw an illuminated image. Then import this two images into the designer. First of all import the dark background image as standard background image. You already know how to do this. Second is to import the illuminated image with

  • 'Image' -> 'Import illuminated image ...'


What does this mean? From now on all the illumination will not be created by the designer. It'll be taken from the imported illumination image. When you add a new illumination frame the content of this frame is cut out from the illumination image. There is no way to mix this up with the designer created illumination.

IMHO there is much more elegant way to import illuminated images, use the illumination snippits.
