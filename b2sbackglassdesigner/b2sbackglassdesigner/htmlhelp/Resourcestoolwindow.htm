The resources tool window - show or hide it with 'F5'.



Shows all involved resources of the backglass project.

  • Export any of these images by right clicking on the image.
  • Select the backglass background image by double clicking on it.
  • Select the illumination image by double clicking on it. Remove all illumination images to get back to the standard illumination mode.
  • Select the 'B2S DMD' image by double clicking on it.
  • Previews all illumination snippits of the project.
