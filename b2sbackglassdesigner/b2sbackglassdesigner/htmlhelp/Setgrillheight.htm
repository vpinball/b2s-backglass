Set grill height

Creating a first B2S backglass ›› More interesting designer stuff ››

To support a two and a three monitor setup a standard B2S backglass is able to show the backglass with or without a grill. Of course if necessary.


Maybe you've started a backglass project with a background image with grill already. If yes please open it. If no please open 'Taxi'. Set the designer's zoom value to about 150% and move the background image that way up that you're able to see the top edge of the grill.


Now start the 'set grill height' mode with ...

  • ... 'Image' -> 'Grill height' -> 'Set grill height'.


Move the mouse cursor over the background image. The cursor is followed by a bright green line. Move the top edge of this line to the top edge of the grill and click the left mouse button. Done. A message box  is completing this action.

If existing a darker green line marks the already set top edge location of the grill height.


To switch off the 'Set grill height' mode without changing the already set grill height click once again on  'Image' -> 'Set grill height'.


That's all you have to do.
