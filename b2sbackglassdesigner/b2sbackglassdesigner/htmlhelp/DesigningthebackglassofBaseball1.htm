Let's return to the three reel displays, for score, homeruns and credits.


There are very interesting infos at the reel displays too. Let's start with the main score display. It's the one at the upper left part of the backglass with a bit thicker orange frame around it. The '1' at the upper left corner of the score frame means this is the reel display with id '1'. This id is very interesting to access this score display, later. 'P1/1' in the upper right corner of the frame shows this display is assigned to 'Player 1' and starts at a overall digit of '1'. And the '32' at the upper right corner is the same as the bulb identifiers, with this identifier you can control the illumination of the reels.


This is the 'Reels & LEDs' tool window for this score display:


A lot stuff that you already know.

Pretty new for you is the 'B2S score type' and the 'B2S player no'. The type is set to 'Scores', the player no is set to 'Player 1'. When you select the type or the player no a recommended value for 'B2S start digit' will be inserted. To set this 'B2S start digit' is the very important part. Here it is set to '1'. Means that this score display starts with digit 1 and since it is made of 4 reels these reels are the reels 1 to 4.

Look at Roger's  'identifier and a backglass counterpart' list. This first reel display matches 'player 1' since it starts with 'B2S start digit' '1':

This list and the wish to access these reels very easily is the reason that

  • reel display 1 starts at 'B2S start digit' '1'
  • reel display 2 starts at 'B2S start digit' '7'
  • reel display 3, the credit reel, starts at 'B2S start digit' '29'


Please notice one thing: Click on the second reel display, the homerun display. Reel display 2 isn't a reel display for player 2 so there isn't the player no set to any value. You will see all the 'relationship' and impact in the next chapter.


Now please click on the 'Reel illumination ...' button of the 'Reels & LEDs' tool window. You will get this window:


Here you can find some well-known infos like 'B2S ID type', 'B2S ID', 'B2S value' and 'Intensity'. They are working like you have seen it the 'Baseball' designing chapter part one or earlier.

With the 'Illu location' you can choose the position of the imaginary light source: From above, from below or from both directions.


Click 'Ok' to save your settings.
