Creating a first B2S backglass

It's time for your first B2S backglass


IMHO the best way to show you how to use the 'B2S Backglass Designer' is to design and create a new B2S backglass.


There is one thing you should know right now: The 'B2S Backglass Designer' is able to create backglasses for EM tables, SS tables or original tables.

EM, Electro-mechanical, pinball machines were manufactured from the 1930's until the mid 1970's, when the transition to SS, solid state machines was made. Original tables aren't a replication of existing machines.

While SS tables are normally running 'Visual PinMAME' to emulate the CPUs and sound chips of the tables EMs and Originals are just controlled by the internal 'Visual Pinball' table script.


I've choosen 'The Bally Game Show', a SS table from 1990. You will learn how to look for some resources, start a new backglass project with all the necessary settings, choose image(s), add some lights and LEDs and complete this first backglass.

While this is a SS table and the communication between table and backglass is done with VPinMAME you can see all the differences between the table types in some later chapters.


Here we go.


  • If this all is absolutely new to you or you want to get infos about some resources please start with the preparation.


  • When your preparation is done or you're familiar with it but you've never created a new backglass please start a new backglass project.


  • You are able to reuse an existing backglass project for a fast entry. Look at the reusing of an existing backglass project.


  • Do you want to add some animations since the standard backglass is boaring? Manage your own animations.



Pretty important to me: Although you have some experience with 'Visual Pinball' and you want to have a quick start and you don't want to read a manual (I know I'm a man too and we do not read any manuals :-)) PLEASE READ THE 'Some interesting tips' chapter. I'm sure you'll find some interesting stuff.
