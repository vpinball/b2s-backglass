Illumination

Tool bar and menu items ››

Show illumination frames

Shows or hides the already added illumination frames in edit mode.


Add a new illumination frame

Adds a new illumination frame.


Add a new illumination snippit

Opens the import dialog for illumination snippits. Here you have to select a file, maybe name it and locate it on the backglass.


Show illumination

Shows or hides the illumination with intensity '2'. To give  you an idea how the backglass will look with this general illumination.


Show illumination with accurate intensity

Shows or hides the illumination with accurate intensity.


Manage animations ...

Opens the dialog to edit animations. Sets the 'Lamp filter' to '(with name)' since only these bulbs can be added to an animation.
