The 'Illumination' tool window - show or hide it with 'F7'.


for 'Bally Game Show' or for 'Baseball'


  • Lamp no: Unique illumination id, set from the designer and not editable
  • Lamp name: Illumination group name. Name grouped lamps with the same name.
  • Initial state: Initial state of the illumination: 'Off', 'On' or 'Always on'. 'On' is on at startup but can be switched off, 'Always on' can not be switched off
  • Dual mode: Illumination assignment to the backglass dual mode (if available), 'Authentic', 'Fantasy' or assigned to 'Both'

  • ROM ID: VPinMAME rom id (only available in settings' 'directB2S' destination type)
  • ROM ID type: VPinMAME rom id type, can be 'Lamp', 'Solenoid', 'GI String' (only available in settings' 'directB2S' destination type)
  • Inverted: Enables or disables the inverted assignment of the illumination to a illu source (only available in settings' 'directB2S' destination type)
    or
  • B2S ID type: B2S id type for data element communication: 'Score 1 Rollover', 'Score 2 Rollover', 'Score 3 Rollover', 'Score 4 Rollover', 'Player Up', 'Can Play', 'Ball In Play', 'Tilt', 'Match', 'Game Over' or 'Shoot Again'. For more information click here.
  • B2S ID: B2S id for data element communication. For more information click here.
  • B2S value:  B2S value for data element communication. For more information click here.

  • Intensity: Illumination intensity, 5 levels.
  • 'Color selector': Illumination color selector.
  • Dodge color: Pretty heavy illumination color effect.

  • Illuminated text: Illuminated text in a bulb.
  • Choose font: Selects the font of the illuminated text.

  • Location: Gets and sets the location of the selected illumination frame.
  • Size: Gets and sets the size of the selected illumination frame.

  • Snippit settings: Opens the following dialog window with illumination snippit settings:
    • Snippit name: The name of the illumination snippit.
    • Snippit type: Type of the illumination snippit: 'Standard image', 'Self rotating image' or 'Mech rotating image'. A self rotating image is just started or stopped by a solenoid and is rotating with the specified count of steps and rotation interval. A mech controlled rotating image is controlled by two solenoids, handled in VP with a so called 'mech' with a special count of steps for a rotation.
    • Z-order: Z-order of overlapping illumination snippits.

    • Mech ID: Id of the controlling mech of the rotation, only available with snippit type 'Mech rotating image'.
    • Steps per rotation: Count of steps for a whole rotation.
    • Rotation steps interval: Timer interval per rotation step, only available with snippit type 'Self rotating image'.
    • Rotation direction: The direction of the rotation, 'Clockwise' or 'Anti-Clockwise'.
