The 'Reels & LEDs' tool window - show or hide it with 'F6'.


for 'Bally Game Show' or for 'Baseball'.


  • Number of players:

  • Display no: Unique display id for the current score display, set from the designer and not editable.
  • Reel/LED count: Count of reels or LEDs of the current score display.
  • Reel/LED spacing: Spacing between the digits of the current score display.
  • Initial state: Initial visibility state of the score display, 'Visible' or 'Hidden'.

  • B2S score type: Score type of the current score display for data element communication: 'Scores' or 'Credits'. For more information click here.
  • B2S player no: Player no of the current score display for data element communication: 'Player 1' to 'Player 6'. For more information click here.
  • B2S start digit: Start digit of the current score display for data element communication. For more information click here.


  • Reel/LED listview: Show all the pre-selected reels or LEDs. To set the current type of a score display click on the score display and then click on an entry of this listview. For more information click here or here.
  • Use 'Dream7' LEDs: Sets the 'Rendered LED' default setting. Recommendation is to use the 'Dream7' LEDs.

  • Location: Gets and sets the location of the selected score display frame.
  • Size: Gets and sets the size of the selected score display frame.
  • Perfect scale, width is fix: Sets the height of the selected score display for a perfect width/height scale.


  • Change LED color: Show an dialog window to change the color of the 'Rendered LEDs' of the current score display.
  • Reel illumination: Sets the illumination settings for reel score displays. For more information click here.
