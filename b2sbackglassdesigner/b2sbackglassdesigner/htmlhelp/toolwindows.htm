Tool windows

Beside the main window there are four tool windows. To open them use the 'Window' menu item. When it's necessary two of them will pop up automatically.


  • 'Resources'
    Open this tool window with 'F5'. This window is showing the basic resources involved in the design process. I.e. there are thumbnail previews for all imported background or DMD images.


  • 'Reels & LEDs'
    Open this tool window with 'F6'. This window is showing all infos about a 'score'. It will pop up automatically if necessary.


  • 'Illumination'
    Open this tool window with 'F7'. This window is showing all infos about a 'bulb'. It will pop up automatically if necessary.


  • 'History'
    Open this tool window with 'F8'. This window is showing a history of your changes. You can undo a lot of your changes especially all moving or resizing actions. But please notice, right now some of the actions are not included in the UNDO mechanism.
