Time to add some lights onto the background image.


For the perfect overview please click the zoom combo box in the main window tool bar to size the image to a perfect window fit. Please notice, of course you are not resizing the image with this functionality, you're just changing the view on it.

After these actions the designer looks like ...


Now add your first bulb. These are the important buttons to add and show some lights:


Click the second button from the left, 'Add a new illumination frame' or use the menu item 'Illumination' -> 'Add a new illumination frame'.

This should be the result:


You have created this white frame over the image. This frame is an illumination frame. Imagine inside this frame is a bulb shining in a perfect circle inside the frame.

The second thing that happened is, the 'Illumination tool window' has opened.


Now use your mouse to move this frame to its final place. Right now this should be over the 'SHOW' letters. Resize the frame for a loose fit.
