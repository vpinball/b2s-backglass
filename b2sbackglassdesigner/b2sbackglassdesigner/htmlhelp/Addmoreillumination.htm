Let's add a lot more illumination frames.


Very important: What you need right now are some external infos. IPDB.ORG is a wonderful source therefor. I found a lot necessary infos in the original manual on the second side (click here and open the 'Operations Manual' PDF) and a great "behind backglass" image (click here for the image) .


We have to start with the manual. What you have to do and please do it very carefully ...

  • Please have a look at the "GAME SHOW Solenoid Table" on side 2 since very often the illumination is controlled by solenoids.
  • Pretty important, you have to look for the terms 'Insert', 'GI' or 'G.I.', 'backbox' or 'backglass'.


This is what we can find:

  • 1B - 'Top Cntr Ln / Insert Teeth Flasher'
  • 2B - 'Rt Back Pnl / Insert 3 Million Flasher'
  • 3C - 'Left Plyfld / Insert Title Flasher'
  • 4C - 'Rt Plyfld / Insert Title Flasher'
  • 6C  - 'Rt Back Pnl / Insert Wheel Flasher'
  • 15  - 'Backbox G.I. Relay'
  • 16  - 'Insert Teeth / Title Flasher'


Well, this means pretty sure, solenoid 1 is controlling a flasher behind the teeth of 'Keith McTeeth', solenoid 2 is doing this behind the 3 million sign, solenoid 3 is doing this job behind the title, and so on. Very interesting is solenoid 15 because this sound like this solenoid is doing a real general illumination job.

As I mentioned earlier, this is a detective's job. I have no idea what's the difference between the two 'Insert Title Flasher'. I have no idea what the letters 'B' and 'C' mean. I have no idea what solenoid 16 is doing with teeth and title once again. And I have no idea whether the lights should go on when the solenoid is firing an 'on' or an 'off'. All this and maybe something more is your 'examination' job.


Okay, let's start with the 'backbox general illumination', called 'G.I.'. Click on the already added illumination frame over "SHOW".

  • Right now there is no need to add a 'Lamp name'.
  • Set the 'ROM ID' to '15' and the 'ROM ID type' to 'Solenoid'.
  • Check the 'Inverted' check box since very often the 'G.I.' is called inverted.
  • Have a look at the 'Intensity', it should be on level '2'.


There is one feature you have to know before adding more frames: The last frame you have clicked and the 'Lamp filter' is providing some values for the next added illumination frame. These means when you add a new illumination frame right now this frame will get the frame size and the 'Intensity' from the last clicked frame and will get the 'ROM' info from the 'Lamp filter'.


The 'Lamp filter' is the combo box between all the illumination buttons of the main tool bar.


Set the 'Lamp filter' to 'IS15'. This is the short form for 'Inverted Solenoid with ID 15'.


Now add the next illumination frame. You see, a lot values of this new frame are already set. Move and size this new frame wherever and how large you want to have it. Repeat this until you have added not more than four or five frames with 'IS15' since it's time to do the first test.


Save your changes and maybe check your illumination.


Time to create the backglass for a very first test.
