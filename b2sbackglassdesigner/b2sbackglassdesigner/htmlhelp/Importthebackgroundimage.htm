Import the background image

Creating a first B2S backglass ›› A B2S backglass from scratch ››

Similar to the start there are two different ways to import a background image:

  • Click on the 'Import background image ...' tool button in the toolbar
  • Click on the menu item 'Image' -> 'Import images' -> 'Import background image ...'.


A file selection dialog will pop up. Navigate to the location of your background image and select it.
