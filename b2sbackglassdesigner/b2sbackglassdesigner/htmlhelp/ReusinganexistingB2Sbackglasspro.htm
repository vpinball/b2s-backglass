Creating a first B2S backglass ››

Okay, your first backglass is done. Do you want to start a new one?


Sometimes is makes perfect sense to start a new backglass project based on an already existing project. Imagine you want to create a backglass for 'Mata Hari' and one of the sample projects is 'Playboy' or you already designed 'Kiss'. All three pins are from 'Bally' and all three are coming from one era. You can save a lot time since normally a lot of the ROM ID connections, general illumination and/or LED displays setup is reused.
