Tool bar and menu items ››

Backglass preview and test ...

Opens a dialog to set some settings for a preview and test start of the backglass. VPinMAME will be started if necessary, therefore you have to select a matching ROM by name.


Export dark backglass image

Exports a dark image of the backglass, all illumination off, into the project sub folder in 'Projects'. Overwrites an already existing file.


Export illuminated backglass image

Exports an image of the backglass with all illumination on, into the project sub folder in 'Projects'. Overwrites an already existing file.


Create 'directB2S' backglass file

Creates the 'directB2S' backglass file in the project sub folder in 'Projects'. Overwrites an already existing file. For more information please click here.


Create 'MS Visual Studio 2010' backglass code

Creates the Microsoft Visual Studio 2010 solution code for a backglass project. Overwrites already existing files but reengineers code changes done in three sections in the Visual Studio solution.
