There is a lot more interesting stuff in the designer.


Do you want to try something without 'killing' your current project state? Create a backup of your project file and you're able to return to this state with just two clicks.


Do you still want to paint your own illuminations with Photoshop or Paint.NET and use it? Two ways, design an illuminated copy of the background image or design small illuminated parts, snippits, of the background image.


You're designing a backglass with a grill? It's pretty simple so don't forget to set the grill height.
