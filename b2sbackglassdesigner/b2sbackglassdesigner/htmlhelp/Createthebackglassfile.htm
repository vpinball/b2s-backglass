Create the backglass file

Creating a first B2S backglass ›› A B2S backglass from scratch ››

Now you have to create the 'directB2S' backglass file.


Click on the menu item ...

  • ... 'Backglass' -> 'Create 'directB2S' backglass code file'.


A new folder is created in the backglass designer EXE folder in the 'Projects' folder. This folder is called like the 'Project name' of the project settings. The backglass file is named like the 'Backglass file name' of the project settings.


Copy the file 'The_Bally_Game_Show_VP911_1.02FS.directb2s' to your 'VP tables folder', edit the 'CreateObject' line of the VP script code, set the line to

'Set Controller = CreateObject("B2S.Server")'

and start the table. The backglass should be loaded for the first time.


Watch the behaviour of the lamps. You have to decide whether the 'Inverted' is set correctly or should be removed. This decision is totally up to you.



And now move ahead and design your first backglass. Have a look at the "behind backglass" image of ipdb.org. Add some illumination frames for

  • the teeth,
  • the wheel,
  • the 3 million and
  • the title flasher.

Get some youtube videos where you can watch the backglass. Arrange the illumination frames like in the original backglass or how you like it.



FYI: I have added the 'Bally Game Show' backglass as a sample backglass. In this sample I have added 64 illumination frames for 'IS15', one for 'IS1', two for 'IS3' and 'IS4', one for 'IS6' and one for 'S2'.
